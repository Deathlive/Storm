# Termux primary setting

Script that performs the initial configuration of the Termux and install Metasploit Framework, RouterSploit, Reaver...


### Installation

```
apt install git
git clone https://github.com/Deathlive/Storm
chmod -R 700 $HOME/Storm
cd Storm
bash start.sh
```
